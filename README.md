# Freelancer-lite

A professional, full-stack marketplace application designed to connect Job Posters with skilled Workers. This platform features a dual-login architecture, localized financial modules, and real-time project tracking.

## 🚀 Key Features
* **Dual Dashboard Logic:** Dedicated views for 'Job Posters' and 'Workers' with dynamic context-switching.
* **Project Life-Cycle Tracking:** Management for jobs through states: Open, In Progress, Completed, and Cancelled.
* **Deadline Alert System:** Automated visual alerts (Urgent/Critical) based on project timestamps to help users prioritize tasks.
* **Financial Module:** Wallet system, secure escrow, and payment history localized in **INR (₹)**.
* **Profile System:** Dynamic cross-profile viewing with resume uploads (PDF/DOCX) and skill badges[cite: 1].
* **Interactive UI:** Responsive sidebar navigation, professional modals, and a built-in messaging interface[cite: 1].

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3 (Custom Flexbox/Grid), Vanilla JavaScript[cite: 1].
* **Backend:** Node.js / Express (or Python)[cite: 1].
* **Database:** PostgreSQL[cite: 1].
* **Security:** Robust API error handling and CORS configurations[cite: 1].
